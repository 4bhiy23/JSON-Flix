# Changelog

All notable changes to this project will be documented here.  

## [1.2.0] - 2025-11-01
### Added
- Redesigned and modernized the Navbar using TailwindCSS.
- Implemented glassmorphism and gradient background for a cinematic feel.
- Added neon hover effects and animated mobile menu.
- Integrated sticky header behavior with backdrop blur and border styling.

### Changed
- Improved overall UI consistency to match the JSON-Flix theme.
- Enhanced color palette with cyan-blue accents for better visual depth.
- Refined responsive transitions and mobile menu interaction.
